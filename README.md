Spotify Genre Grouping and Music Recommendation
Technologies: Python, Pandas, NumPy, Scikit-learn, Matplotlib

Built a K-Means clustering model to group Spotify songs based on audio features.
Applied data cleaning, feature scaling, and exploratory data analysis for clustering.
Used the Elbow Method to identify the optimal number of clusters and visualized results.
Developed a recommendation feature that suggests similar songs from the same cluster.
